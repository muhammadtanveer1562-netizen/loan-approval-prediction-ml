# 🏦 Loan Approval Prediction System

An end-to-end Machine Learning project that predicts whether a loan application will be **Approved** or **Rejected** using classification algorithms.

This project was developed as part of my Machine Learning Internship at Elevvo.

---

## 📌 Project Description

The goal of this project is to build a binary classification model that predicts loan approval status based on applicant financial and personal details.

The dataset includes features such as:

- Number of dependents  
- Education level  
- Self-employment status  
- Annual income  
- Loan amount  
- Loan term  
- CIBIL score  
- Asset values  

Target Variable:
- `1` → Loan Approved  
- `0` → Loan Rejected  

---

## ⚙️ What This Project Includes

- Data cleaning & preprocessing  
- Handling missing values  
- Encoding categorical variables  
- Feature scaling  
- Handling imbalanced data using SMOTE  
- Logistic Regression & Decision Tree comparison  
- Evaluation using Precision, Recall, and F1-score  
- Streamlit web application for real-time prediction
